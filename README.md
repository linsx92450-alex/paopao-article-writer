# paopao-article-writer

> 一个用于 Claude Code 的 Skill，模仿"水球泡"（泡泡）的抖音短视频风格，生成可直接用于口播逐字稿或公众号的中文文章。

## 简介

`paopao-article-writer` 基于 Claude Code Skill 系统构建，帮助用户快速生成具有"泡泡风格"的短视频文案或公众号文章。

泡泡（@水球泡）是抖音上的内容创作者，以犀利、接地气、循循善诱的表达风格著称。本 Skill 提炼了她的核心写作逻辑，并针对 AI 生成内容常见的两个问题做了专项优化：

- **断句碎、AI 味重**：强制要求句子连贯，一个意思一口气说完，不是一句话停顿一下再说下一句
- **说教意味浓**：不把结论直接喂给读者，而是带读者走一遍思路，让他自己在最后想明白

## 适用场景

- 抖音 / 视频号短视频口播逐字稿
- 微信公众号观点类文章
- 职场、AI、创业、求职、社会观察等议题的内容创作

## 安装

**全局安装**（所有项目均可使用）：

```bash
git clone https://github.com/your-username/paopao-article-writer.git
cp -r paopao-article-writer ~/.claude/skills/
```

**项目级安装**（仅对当前项目生效）：

```bash
cp -r paopao-article-writer /your/project/.claude/skills/
```

安装后重启 Claude Code，Skill 自动生效。

## 使用方法

在 Claude Code 对话中，直接用自然语言触发：

```
帮我写一篇关于职场内卷的文章
用泡泡风格写一篇 AI 会不会取代产品经理
写一篇关于创业心态的短视频脚本
泡泡文章生成：及格心态是最贵的策略
```

## 风格核心

### 反常识开头

第一句必须是让人"等等，什么？"的断言，直接亮刀，不铺垫。

### 循循善诱，不说教

不用"你应该"、"你需要"，不在结尾列行动清单。带读者走一遍思路，让他在某个时刻自己说"原来如此"。

### 句子连贯，像真人说话

同一意思的句子放在同一段落，用"——"、"其实"、"但"等词衔接，形成完整气口，不是 AI 式的一句一断。

### 案例要有名字、细节、情绪

案例不是用来"举例说明道理"的，是让读者看到一个真实的人，感受到"天哪，这不就是我"。

### 情绪要有起伏

有点气、带点讽刺、带着心疼——不是表演情绪，是真的有情绪。不要每句话都保持同一个温度。
